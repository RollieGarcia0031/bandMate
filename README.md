# BandMate - Connect, Collaborate, Create Music

BandMate is a modern web application designed to connect musicians, fostering collaborations, and helping them find their next bandmates or musical projects. Inspired by the intuitive "swipe" mechanics of popular dating apps, BandMate offers a unique and engaging experience for musicians to discover compatible artists based on their skills, genres, instruments, and musical preferences.

Whether you're looking for a drummer for your rock band, a guitarist for a jam session, or a vocalist for a new recording project, BandMate makes it easy to find and connect with the right people.

## Features

-   **User Authentication:** Secure login and registration for musicians.
-   **Musician Profiles:** Create detailed profiles showcasing your instruments, genres, skill level, and musical influences.
-   **Discovery & Matching:** Intuitively swipe through profiles of other musicians, indicating your interest.
-   **Real-time Messaging:** Connect and chat with musicians you've matched with to discuss potential collaborations.
-   **Interactive Landing Page:** A compelling introduction to BandMate with a hero section, features overview, how-it-works guide, and testimonials.
-   **Responsive Design:** Seamless experience across various devices and screen sizes.

## Technologies Used

BandMate is built with a robust and modern tech stack, ensuring a fast, scalable, and maintainable application:

-   **Next.js:** A React framework for building performant web applications with server-side rendering and static site generation capabilities.
-   **React:** A JavaScript library for building user interfaces.
-   **TypeScript:** A typed superset of JavaScript that compiles to plain JavaScript, enhancing code quality and developer experience.
-   **Tailwind CSS:** A utility-first CSS framework for rapidly building custom designs.
-   **Shadcn UI:** A collection of beautifully designed components built using Radix UI and Tailwind CSS, providing a consistent and accessible user interface.
-   **Supabase:** An open-source Firebase alternative providing a powerful PostgreSQL database, authentication, real-time subscriptions, and storage.
